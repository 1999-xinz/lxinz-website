# 阶段 05 - 启动动画终端化

## 目标

将启动动画改造成更贴近参考图的终端启动效果，并确保每次刷新页面都重新播放。

## 已完成

- 去掉了基于 `sessionStorage` 的“仅首次播放”逻辑。
- 改为每次刷新都展示启动动画。
- 启动动画改为终端风格逐行逐字打印。
- 文案调整为更贴合当前项目的 `pnpm`、博客、canvas、交互实验语境。
- 增加终端扫描线、网格背景、绿色荧光光晕和闪烁光标。
- `Skip Animation >>` 入口固定在右上角。

## 相关文件

- `src/components/intro-gate.tsx`
- `src/index.css`

## 说明

这一阶段仍然是 CSS 驱动的启动动画，没有接入真实的 canvas 渲染，因此后续还可以继续升级成更强的动态版本。
