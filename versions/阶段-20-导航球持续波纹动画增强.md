# 阶段-20：导航球持续波纹动画增强

## 目标

为右上角导航球（`FloatingOrbNav`）增加持续播放的环形波纹动画，并在第二次调优中提升每圈波纹宽度与视觉强度，让入口更有动势感。

## 本阶段改动

### 1) 新增持续循环波纹（第一版）

在 `.orb-nav` 上增加双伪元素波纹层：

- 使用 `::before` 与 `::after` 两层圆环
- 两层共享同一套扩散动画 `orb-ripple`
- 通过 `animation-delay` 做半周期错峰，形成“连续不断”的波纹节奏

关键实现点：

- 抽离尺寸变量 `--orb-size`（桌面 `76px`，移动端 `68px`）
- 波纹起始与球体同心（`translate(-50%, -50%)`）
- 动画从 `scale(1)` 扩散到 `scale(1.6)`，并伴随透明度衰减

### 2) 波纹加宽与视觉增强（第二版）

根据效果反馈“每圈波纹太窄”，进一步增强：

- 波纹边框由 `1px` 提升到 `2px`
- 提高边框可见度（透明度增强）
- 增加轻微外发光 `box-shadow`，强化扩散存在感

## 结果

- 导航球外圈具有稳定、连续的呼吸式波纹效果
- 每圈波纹更厚、更清晰，视觉识别度更高
- 不影响原有导航球开合与面板交互逻辑

## 涉及文件

- `src/styles/_navigation.scss`

## 备注

本阶段仅涉及样式层，无组件结构和路由逻辑变更。后续如需继续增强，可在保持节奏的前提下微调：

- 扩散终点比例（如 `1.6 → 1.8`）
- 动画周期（如 `2.8s`）
- 发光强度与透明度参数
