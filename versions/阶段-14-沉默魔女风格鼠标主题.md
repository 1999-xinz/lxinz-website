# 阶段 14 - 沉默魔女风格鼠标主题

## 目标

参考提供的概念图，制作一套更接近“沉默魔女”主题气质的网页鼠标指针族，并应用到项目中。

## 已完成

- 将原有手套鼠标替换为新的主题光标资源。
- 新增 `default / pointer / active / help / not-allowed / text / move` 七类语义光标。
- 默认箭头改为米白描边、金棕主体、绿色柄线的主题箭头。
- 指向与按下态改为更接近参考图语境的手型交互光标。
- 帮助、禁止、文本、移动等状态补齐为可复用主题变量。
- 全局样式规则切换为基于语义光标变量驱动，方便后续页面复用。

这次实现是基于参考图进行网页光标化改造，目标是高度接近主题气质并适配浏览器限制，而不是照搬桌面系统光标文件格式。
