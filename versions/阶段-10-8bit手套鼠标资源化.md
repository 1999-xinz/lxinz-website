# 阶段 10 - 8bit 手套鼠标资源化

## 目标

将鼠标样式从内联 data URI 方案升级为更接近游戏 UI 的 8bit 手套手，并尽量覆盖常见光标状态。

## 已完成

- 新增独立光标资源文件，替代原先直接写在样式中的 data URI。
- 默认光标替换为更大的 8bit 手套手。
- 为悬停/可点击状态单独设置手套指向光标。
- 为按下状态单独设置按压版光标。
- 扩大覆盖范围，补充 `button`、`a`、`summary`、`label`、`select`、常见交互型 `input`、`role` 和 `tabindex` 元素的光标规则。
- 为文本输入类场景保留文本光标。

- `public/cursors/glove-pointer.svg`

## 说明

推荐使用独立图片资源而不是继续堆在 data URI 里，因为后续更换风格、调热点、补不同状态都会更好维护。
