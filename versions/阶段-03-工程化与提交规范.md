# 阶段 03 - 工程化与提交规范

## 目标

为项目建立基础格式化、提交校验与 Git hooks。

## 已完成

- 增加 `Prettier` 配置。
- 增加 `format`、`format:check`、`typecheck`、`lint` 脚本。
- 增加 `lint-staged` 配置。
- 接入 `Husky` 的 `pre-commit` 与 `commit-msg`。
- 接入 `commitlint` 的 conventional commits 校验。

## 相关文件

- `.husky/pre-commit`
- `.husky/commit-msg`
- `.prettierrc.json`
- `.prettierignore`
- `commitlint.config.cjs`

## 说明

当前校验主链以 `Prettier + TypeScript typecheck` 为主，优先保证提交过程稳定。
